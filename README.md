# AI-Driven Optimization of 5G Resource Allocation

Overview
This project applies machine learning to optimize resource allocation in 5G networks.

The model predicts optimal bandwidth allocation based on:
Application type

Signal strength
Latency requirements
Required bandwidth
Allocated bandwidth

The system aims to improve:
Network efficiency
User experience
Resource utilization

🎯 Problem Statement

5G networks support a wide range of applications including:
streaming
gaming
IoT devices
enterprise systems

Traditional resource allocation methods struggle to manage dynamic traffic patterns and QoS requirements.
This project addresses the challenge by building an AI-driven allocation system.

🧠 Machine Learning Workflow

Data Processing:

Data collection
Data cleaning
Feature engineering
Normalization

Model Training:

Multiple algorithms were evaluated including:
Logistic Regression
Decision Tree
Random Forest
Linear Regression
The best-performing model was selected based on prediction accuracy.

Users enter parameters including:
Application Type
Signal Strength
Latency
Required Bandwidth
Allocated Bandwidth

The model predicts:
Optimal resource allocation percentage.

System components include:
User Interface

Users interact with a web interface that sends parameters to the ML model and returns allocation predictions.
